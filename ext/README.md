# External Files

## Visual Studio 2017

To build this project add the following dependencies:

* [rgm-0.1.1.zip](https://github.com/rioki/rgm/releases/download/v0.1.1/rgm-0.1.1.zip)
* [SDL2-devel-2.0.7-VC.zip](http://libsdl.org/release/SDL2-devel-2.0.7-VC.zip)
* [SDL2_image-devel-2.0.1-VC.zip](https://www.libsdl.org/projects/SDL_image/release/SDL2_image-devel-2.0.1-VC.zip)
* [SDL2_ttf-devel-2.0.14-VC.zip](https://www.libsdl.org/projects/SDL_ttf/release/SDL2_ttf-devel-2.0.14-VC.zip)

Just copy the files here and unzip them in place.
